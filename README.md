# Speech-to-Reasoning-Pipeline-with-Whisper-and-a-Quantized-LLM
An end-to-end speech-to-reasoning pipeline using OpenAI Whisper and a 4-bit quantized LLM (DeepSeek-R1-Distill-Llama-8B via Unsloth). Built for Google Colab, it features automated audio transcription, text cleaning, sequential GPU memory management, and step-by-step chain-of-thought reasoning responses.
